# HomeBlessedHome

1.	Descrição
	Site para postar pequenos blocos de texto com imagem mostrando o dia-a-dia da dona de casa.

2.	Tecnologia
	Utilizar: 
	•	Bootstrap-3.3.6
	•	jQuery-1.11.4
	•	Codeigniter-3.0.6

3.	Páginas
	•	Home 
		o	Modal 
			?	Login
			?	Mudar Configuração
			?	Adicionar Post
			?	Apagar Post
		o	Header
			?	Titulo
			?	Descrição
			?	Data
		o	Posts em Blocos

4.	Banco de Dados
	•	Configuração
		o	Titulo
		o	Descrição
		o	Expiração padrão em dias
	•	Posts
		o	Data/Hora
		o	Imagem (filename=datetime_post_id)
		o	Tags
		o	Texto (MAX:300)
		o	Usuario
	•	Usuario
		o	Nome
		o	Login
		o	Pass
